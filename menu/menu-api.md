## API

### \<u-menu\>

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| `[uDatas]`      | 绑定的数据     | `UMenuNode[]` | |
| `[uCustomNode]` | 自定义节点模板 | `TemplateRef<UMenuNode>` | |
| `[uMode]` | 模式 | `'vertical' \| 'horizontal' \| 'inline'` |  `'inline'` |
| `(uNodeClick)`  | 点击节点事件   | `EventEmitter<UMenuFlatNode>` | |
