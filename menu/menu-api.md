## API

### u-menu

| 参数           | 说明           | 类型                          | 默认值 |
| -------------- | -------------- | ----------------------------- | ------ |
| `[datas]`      | 绑定的数据     | `UMenuNode[]`                 |        |
| `[customNode]` | 自定义节点显示 | `TemplateRef<UMenuNode>`      |        |
| `(nodeClick)`  | 点击节点事件   | `EventEmitter<UMenuFlatNode>` |        |
